# eshop app deplyoment

1. Create namespace
kubectl create namespace eshop

2. Install postgres
helm repo add bitnami https://charts.bitnami.com/bitnami
helm install eshop -n eshop -f ./postgres/values.yml bitnami/postgresql

3. Install kafka
helm install eshop -n eshop -f ./kafka/values.yml bitnami/kafka

4. Apply configmap and secrets
kubectl apply -f ./app/config.yml

5. Apply database migration
kubectl apply -f ./app/migration-job.yml

6. Deploy application
kubectl apply -f ./app/deployment.yml
